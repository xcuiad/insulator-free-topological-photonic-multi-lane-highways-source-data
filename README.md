# Source Data for Insulator-free Topological Photonic Multi-Lane Highways

This repository contains the source data used to generate the main-text and extended-figure panels for the Nature paper "Insulator-free topological photonic multi-lane highways".

The repository is organized by figure family and panel group. The `.xlsx` files are the data intended for upload. Mathematica notebooks were previously kept alongside the data as plotting scripts, but they are not part of the upload set.


## Included Data

- `Main-Figs/`: source data for main-text figures
- `Extended-Figs/`: source data for extended figures
- File type: `.xlsx`
- Current upload set: 29 spreadsheet files

## Excluded Files

- Local operating-system files such as `.DS_Store`

## How to Use

Open the `.xlsx` files with Excel, LibreOffice, or another spreadsheet viewer/editor. The filename usually matches the corresponding figure panel group.

## Citation

If you use this data, please cite the associated paper and this repository. The machine-readable citation metadata is in `CITATION.cff`.
